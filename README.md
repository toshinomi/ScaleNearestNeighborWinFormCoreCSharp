# ScaleNearestNeighborWinFormCoreCSharp

# 概要
【 ScaleNearestNeighborWinFormCoreCSharp 】  
最近傍補間法を使用した画像の拡大・縮小アルゴリズムの Windows Form アプリケーションです。  
デザインはフラットデザインにしています。  

開発環境：Visual Studio 2019  
言語：C#、.NET Core 3.0
